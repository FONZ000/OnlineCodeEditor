About G-EDITOR

×

Welcome to G-EDITOR

G-EDITOR is an online code editor that supports only three languages for now.

  • HTML
  • CSS
  • JAVASCRIPT

This code editor can make learning the above mentioned languages easier to learn, it can be used in creating a similar website to GitHub or stack overflaw & a CMS. This code Editor supports the following features:

  • Syntax Highlight.
  • AutoCompletion.
  • HTML mixed language mode.
  • You can use cdnjs css & js libraries like Bootstrap.
  • Search for text inside your code.
  • Replace keywords inside your code.
  • Auto-close brackets & Tags.
  • Display errors & warnings.
  • Fold Gutters.
  • Show Active line.

Hope you enjoy my work.
